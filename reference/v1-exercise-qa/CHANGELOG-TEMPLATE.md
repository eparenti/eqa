## Changelog Entry Guidelines

### Categories
- **Added** - New features
- **Changed** - Changes in existing functionality
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Vulnerability fixes

### Entry Format
```
- Brief description of change [#issue-number] (@author)
```

### Examples
```
### Added
- New scale-review exercise for Chapter 6 [#PTL-1234] (@dspohn)
- Support for host patterns in exercises [#PTL-1235] (@miphilli)

### Fixed
- Corrected cleanup playbook to remove all artifacts [#PTL-1236] (@mikelly)
- Fixed idempotency issue in scale-files exercise [#PTL-1237] (@ragrawal)

### Changed
- Updated to AAP 2.5 compatibility [#PTL-1238] (@mikelly)
```
